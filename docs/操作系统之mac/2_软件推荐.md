# 截图软件

## snipaste(推荐)

细节功能最多的。比如：通过快捷键微调截屏区域大小等，这些功能一般不怎么用得上但是需要时肯定是很实用的

```
 brew install --verbose --cask snipaste
```

## ishot

功能比较多，可以滚动截图、区域录屏
