2、登陆 OpenAI 平台，点右上角头像进入，[View API keys](https://platform.openai.com/account/api-keys)

3、点击 Create new secret key，并复制生成的 key，存到自己的设备里面，最好是直接存到苹果设备 Note 里，方便下一步粘贴使用

4、打开苹果设备的 Shortcuts App 和 Notes App
