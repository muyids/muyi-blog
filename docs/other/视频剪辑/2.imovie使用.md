# 快捷键

```
L播放素材      LL 快速播放       LLL更快
J回看素材      JJ                     JJJ
K停止
i视频入点 o视频出点  f设为喜欢（绿线）
delete 删去（红线） u取消拒绝或喜欢
e添加到视频末尾  w插入到浏览头位置
q时间线上方（b-roll）/添加字幕
shift+z 时间线最合适的缩放
shift+command+z 撤销操作
左右箭头可拖拽（原始素材范围内任意延伸）
command+b 切开片段
option+/  拆分+删除
精修片段（小于三帧）英文,缩短视频    .延长视频
r不送+delete 删除中间片段
r option+/
⬆️⬇️ 片段头尾浏览  ⬅️➡️慢帧数浏览
/从头开始播放片段
/+command+L 锁定片段进行播放
option+command 改变时间线上下元素扎点
不送光标+option  字幕或音效重复添加
转场  交叉叠化：command+t
     b-roll:视频上小圆点控制交叉叠化
q添加字幕
r+拖拽选取 可单独设定
shift+command+m 音频静音
统一色调：1.command+c 选定片段色调
2.option+command+c 色调复制到其它片段
ken burns
command+r 改变视频速度 （60帧修减速）
1080p 60帧率  4k分辨率
新手建议1080p 30帧
```
