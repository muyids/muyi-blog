# 指标体系

指标，实际上就是一种度量。

大到用于监控和评估商业进程的状态，小到衡量某个功能模块的情况，或者是自己的活动效果。指标体系的构建，是为了让业务目标**可度量、可描述、可拆解**，所以我们的指标是基于业务目标抽取和评估的数据维度，这些数据为度可以用来帮助达到业务目标。
