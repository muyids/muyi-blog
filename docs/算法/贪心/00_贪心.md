# 贪心

每次都求最优解，以局部最优得到全局最优

贪心是**动态规划的一种特殊类型**的题目

# 概念题

找零钱：有 1，2，5 三种面值的钞票，每件商品售价 3 元，输入顾客付款的钞票面额顺序，问能否找零？

算法思路：优先找较大面值的钱

题目来源：[LeetCode 860. Lemonade Change (easy)](https://github.com/muyids/leetcode/blob/master/algorithms/801-900/860.lemonade-change.md)
